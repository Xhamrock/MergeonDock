MergeonDock

Open-Source Molecular Docking & Analysis Tool

What is MergeonDock?

MergeonDock is an open-source graphical tool designed for molecular docking and interaction analysis. It integrates AutoDock, RDKit, PyMOL, and Open Babel to help researchers streamline their docking workflow without requiring command-line expertise.

Key Features

  • AutoDock Vina & AutoDock 4.2 integration
  • Built-in Open Babel file converter
  • Receptor and ligand preparation with GUI
  • Visualization powered by PyMOL
  • Compatible with .pdb, .pdbqt, and .sdf formats
